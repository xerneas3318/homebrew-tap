# xerneas3318/homebrew-tap

Homebrew tap for [@xerneas3318](https://github.com/xerneas3318)'s tools.

## Install

```sh
brew tap xerneas3318/tap
brew install neurozip          # or: brew install hackfetch
```

You can also install without tapping first:

```sh
brew install xerneas3318/tap/neurozip
```

## Formulae

| Formula | Description |
|---|---|
| [`neurozip`](https://github.com/xerneas3318/NeuroZip) | Task-aware neural compression for EEG: compress brain recordings while keeping them text-searchable |
| [`hackfetch`](https://github.com/xerneas3318/hackfetch) | Hack Club themed system fetch with live Hackatime stats |

## Updating

```sh
brew update
brew upgrade neurozip
```
